# Sample Unity projects of MYTY Kit

# Usage (For Standalone, iOS)
- Open project, Select a suitable target build platform in build settings and Build. That's all!


# Usage (For WebGL)
- Open project
- Select WebGL as target buildplatform. 
- Build project. Please set the output name to `webbuild`
- Copy the `webbuild` directory to the `web/public` in WebGL project directory
- Open terminal and go to `web` directory
- Run the sample react project in `web` following the README in the directory.
